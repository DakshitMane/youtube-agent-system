PROJECT SUMMARY: YouTube Agent System - AI-Powered Video Generator
=====================================================================

OVERVIEW
────────
The YouTube Agent System is a fully automated, multi-agent AI platform that 
generates professional YouTube videos with animated slides, topic-specific 
scripts, and quality validation. Built with Python, it orchestrates multiple 
specialized AI agents to handle research, writing, production, and quality 
assurance automatically.

CORE CAPABILITIES
─────────────────

1. MULTI-AGENT ORCHESTRATION
   • Parallel research agents (trends, facts, competition)
   • Sequential script writing agent
   • Iterative quality validation agent
   • Central production team orchestrator
   • Agent-to-Agent (A2A) communication protocol

2. INTELLIGENT CONTENT GENERATION
   • Topic-aware script creation
   • Automatic outline generation
   • Dynamic pacing based on video duration
   • Key point extraction
   • SEO-optimized structure

3. ANIMATED SLIDE GENERATION
   • Gamma-style professional presentations
   • Multiple transition types (fade, slide, wipe)
   • Text animations (typewriter, staggered fade-in)
   • Gradient backgrounds
   • Decorative animated elements
   • Full HD 1920x1080 @ 30 FPS

4. QUALITY ASSURANCE
   • 5-iteration improvement loops
   • Component-based quality scoring
   • Automatic content refinement
   • Quality metrics (0-1.0 scale)
   • Session tracking and monitoring

TECHNICAL STACK
───────────────
• Language: Python 3.11+
• Video Processing: OpenCV (cv2)
• Image Processing: PIL/Pillow
• Numerical Computing: NumPy
• Data Validation: Pydantic
• Async Programming: Python asyncio
• Configuration: Python-dotenv

Add Custom Tools:

Add Custom Content:

  Add topic keywords and templates
  Implement specific script generation

TROUBLESHOOTING
───────────────

Issue: OpenCV import error
Solution: pip install opencv-python

Issue: Slow video generation
Solution: System is CPU-bound. Use faster hardware or reduce duration.

Issue: Memory issues
Solution: For very long videos, generate shorter videos and concatenate.

Issue: Content not topic-specific
Solution: Update topic handling in sequential_agents.py
